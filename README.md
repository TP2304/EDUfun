# EDUfun

**EDUfun** is an educational quiz application designed for elementary school students to learn and practice knowledge in a fun and interactive way.

---

## Overview

The app offers quizzes in **four categories**: History, Geography, Music, and Technology.  
Each category contains **10 questions**, with **4 multiple-choice answers** per question. A timer runs for each question ‚è±, adding a sense of challenge and urgency.
